#LIVE
Generate using OpenSSL: Run in terminal
1. Create private key
openssl genpkey -algorithm RSA -out private_key.pem -pkeyopt rsa_keygen_bits:2048
2. Create public key
openssl rsa -pubout -in private_key.pem -out public_key.pem
